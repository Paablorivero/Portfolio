# 🌌 Portfolio Personal | Pablo Rivero

> *"De los circuitos al código"* ⚡

Este repositorio contiene el código fuente de mi portfolio personal. Diseñado y desarrollado desde cero con un enfoque minimalista, utilizando **Astro** para un rendimiento óptimo y **CSS nativo** para un diseño estructurado y elegante.

🔗 **[Ver el Portfolio en vivo](TU_ENLACE_AQUI)** *(Añade aquí la URL cuando lo publiques)*

---

### ✨ Características Destacadas

* **Diseño Minimalista y Dark Mode:** Interfaz limpia con fondo negro y tipografías cuidadas que ceden el protagonismo al contenido y la imagen personal.
* **Layout Interactivo:** Navegación posicional distribuida en las esquinas de la pantalla mediante el uso avanzado de `position: absolute` en CSS.
* **Rendimiento Extremo:** Construido con Astro, generando HTML estático ultrarrápido sin sobrecargar el navegador con JavaScript innecesario.
* **Diseño Responsive:** Perfectamente adaptado para funcionar con fluidez tanto en escritorio como en dispositivos móviles.
